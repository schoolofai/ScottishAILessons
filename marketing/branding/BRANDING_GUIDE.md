## Brand Guidelines Summary

### Do's ‚úÖ
- Always mention SQA alignment when describing curriculum
- Use Scottish examples (¬£, local contexts) in content
- Highlight accessibility features prominently
- Show real student mastery data (anonymized)
- Emphasize evidence-based learning science
- Use inclusive language ("learners" not just "students")
- Showcase interactive diagrams visually
- Mention self-hosting option for schools

### Don'ts ‚ùå
- Don't claim to replace teachers (you're an assistant)
- Don't use generic UK curriculum examples
- Don't hide accessibility behind optional features
- Don't overpromise AI capabilities ("guaranteed A grades")
- Don't ignore data privacy concerns
- Don't use complex jargon without explanation
- Don't show static screenshots when interactive demos available
- Don't lock core features behind premium tiers (accessibility must be free)
